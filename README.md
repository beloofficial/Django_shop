# Carrito de compras con el framework Django
Esta es una aplicación web hecha con [Django](https://www.djangoproject.com/)Django + [Python](https://www.python.org/).

## Caracteristicas
- Registrar productos
- Insertar cupones de descuento
- Authenticacion y autorizacion
- Mostrar productos por categorias
- Realizar ordenes de compra
- Carrito de compras

## Requisitos
```sh
# instalar Django 
$ pip install Django==1.8.6

# instalar Pillow
$ pip install Pillow==2.9.0

# instalar widget-tweaks
$ pip install django-widget-tweaks
```

## Requisitos
Instalar virtualenv, instalar los requisitos y luego:
```bash
python manage.py runserver
```

## Versiones
- Django 1.8.6
- Python 3.5
